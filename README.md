# Etch-a-Sketch

A digital Etch-a-Sketch web application built with HTML, CSS, and JavaScript. This project recreates the classic drawing toy experience in your browser, allowing you to create colorful pixel art by hovering over a grid.

## Features

- **Interactive Grid**: Hover over squares to change their color to random RGB values
- **Customizable Size**: Create grids from 1x1 up to 100x100 squares
- **Responsive Design**: Grid scales appropriately on different screen sizes
- **Clean Interface**: Simple, intuitive user interface with a single control button

## How to Use

1. Open `index.html` in your web browser
2. The page loads with a default 16x16 grid
3. Hover your mouse over the squares to draw with random colors
4. Click the "Create new grid" button to specify a different grid size
5. Enter a number between 1 and 100 when prompted

## Technologies Used

- HTML5
- CSS3 (with CSS custom properties for dynamic grid sizing)
- Vanilla JavaScript (ES6+)

## Project Structure

- `index.html` - Main HTML structure
- `style.css` - Styling and responsive layout
- `script.js` - Interactive functionality and grid generation

## Live Demo

You can view a live demo by opening the `index.html` file in any modern web browser.
